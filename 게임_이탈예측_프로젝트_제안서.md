# 게임 유저 이탈 예측 프로젝트 제안서

## 1. 프로젝트명
게임 유저 이탈 예측에서 머신러닝과 다층 퍼셉트론(MLP) 성능 비교 및 주요 영향 변수 분석

## 2. 연구 배경 및 필요성
온라인 게임 서비스에서 유저 이탈은 운영 효율성과 수익성에 직접적인 영향을 미친다. 특히 최근에는 단순 통계 기반 관리보다 데이터 기반 예측 모델을 활용한 선제 대응의 중요성이 커지고 있다. 본 프로젝트는 게임 유저 행동 데이터를 기반으로 이탈 여부를 예측하고, 어떤 행동 특성이 이탈에 영향을 주는지 분석하여 실무적으로 해석 가능한 인사이트를 도출하는 것을 목표로 한다.

## 3. 연구 목적
본 프로젝트의 목적은 다음과 같다.
1. 유저 행동 특성을 바탕으로 이탈 여부를 예측하는 분류 모델을 구축한다.
2. 머신러닝 모델(Logistic Regression, Random Forest)과 딥러닝 모델(MLP)의 성능을 비교한다.
3. 변수 영향 분석을 통해 이탈에 중요한 요인을 해석한다.
4. 결과를 바탕으로 이탈 방지 전략 수립에 활용 가능한 시사점을 제시한다.

## 4. 데이터셋 및 변수 구성
본 연구는 라벨이 포함된 중소형 탭울러(tabular) 데이터셋을 사용한다. 데이터는 공개 데이터셋(Kaggle 등) 또는 수업에서 허용한 범위의 유저 행동 로그 형식을 따른다.

- 입력 변수(예시)
  - 주간 접속 일수
  - 평균 플레이 시간
  - 최근 결제 금액
  - 최근 승률
  - 소셜 활동 지표(친구 수, 파티 횟수)
  - 최근 7일 활동량 변화율
- 타깃 변수
  - 이탈 여부(예: 향후 7일 미접속 시 1, 아니면 0)

## 5. 연구 방법
프로젝트는 문제 정의, 데이터 분석, 모델링, 결과 해석의 순서로 진행한다.

### 5.1 데이터 분석(EDA)
- 클래스 비율(이탈/비이탈) 확인
- 변수 분포 및 이상치 확인
- 상관관계 히트맵 분석
- 이탈 그룹과 비이탈 그룹 간 분포 비교(Boxplot)

### 5.2 전처리
- 결측치 처리
- 범주형 변수 인코딩
- 수치형 변수 스케일링(StandardScaler)
- 학습/검증/테스트 데이터 분할

### 5.3 모델링
다음 3개 모델을 동일한 데이터 분할 기준에서 학습하고 성능을 비교한다.
1. Logistic Regression
2. Random Forest
3. Multi-Layer Perceptron(MLP)

MLP는 은닉층, 활성화 함수(ReLU), 손실 함수(Binary Cross-Entropy), 최적화 알고리즘(Adam)을 사용하며, 오차역전파와 경사하강법의 개념을 학습 과정과 함께 설명한다.

### 5.4 성능 평가 및 해석
- 평가 지표: Accuracy, Precision, Recall, F1-score, ROC-AUC
- 혼동행렬을 통한 오분류 패턴 확인
- Random Forest 중요도 또는 Permutation Importance를 통한 변수 영향 분석
- 필요 시 SHAP 기반 해석 확장

## 6. 기대 효과
1. 이탈 위험 유저를 조기에 식별하는 예측 프레임워크를 제시할 수 있다.
2. 머신러닝 대비 딥러닝(MLP)의 성능 및 한계를 비교 검토할 수 있다.
3. 모델 결과를 해석 가능한 형태로 제시함으로써 실무 적용 가능성을 높일 수 있다.

## 7. 추진 일정(예시)
- 1주차: 데이터셋 확정 및 문제 정의
- 2주차: EDA 및 전처리
- 3주차: 기준 모델(Logistic Regression, Random Forest) 구축
- 4주차: MLP 모델 설계 및 학습
- 5주차: 성능 비교 및 변수 영향 분석
- 6주차: 결과 정리, 발표 자료 및 최종 보고서 작성

## 8. 결론
본 프로젝트는 게임 유저 이탈 예측 문제를 대상으로 머신러닝과 딥러닝을 비교하고, 단순 성능 비교를 넘어 변수 영향까지 분석하는 것을 목표로 한다. 이를 통해 수업에서 요구한 다층 퍼셉트론, 오차역전파, 경사하강법, 결과 해석 요소를 통합적으로 충족하는 실용적 프로젝트를 수행하고자 한다.
